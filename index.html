Description of the Dataset

I obtained this data by requesting it from my Twitter archive. It ranges from mid-2019 to November 2023. There is a period that seems to be ‘missing‘ in the data, but it is organically missing. This means that it was not present in the archive to begin with. The reason for this is that I had deleted a lot of my twetes of that periodon my own a couple of years ago.

The dataset consisted of around 12800 tweets and retweets. The raw data can not be disclosed for privacy concerns, but each tweet had an associated number of likes, number of retweets, classified language, hashtags and mentions. Another set of data I used was that concerned with advertisements which consisted of around 24000 instances of advertisements over the period of me having Twitter, approximately 7 years.

This data can be found in the GitHub repository for this website under /data.

Mode Hour

Over time, the hour during which I tweeted the most changed. However, there's still a peak that remained pretty much the same: evening.

This is a very detailed graph that shows the hour during which I tweeted most for every day in the dataset. Often, the peaks would be around the same time for consecutive days, given that I am a person who has a routine. Because of my routine, it would make sense that most tweets are either early morning or around evening.

We can see some patterns that map to events in my life. During my A Level exams, there were many fluctuations in my routine, which can be seen in the visible May-June periods in 2021 and 2022. During summers, we'll have more points during the early hours of the morning in comparison to winter. In winter 2022, I would never be up before 06:00, so all of the points are away from that time.

This is one of the more informative graph in this analysis thus far. Here are some takeaways:

1. I use Twitter on a daily basis.

2. For a number of consecutive days, the hour during which I tweet most, denoted as "mode hour", are close. This means that I keep a somewhat stable routine. This does not perfectly apply to the first half of the dataset, when I was doing my A Levels and had a very simultanous routine unlike when I started university and established a routine, somewhat. Again, comparison with my Google Calendar data will bring a lot more depth to this analysis.

3. My peak activity would be either early morning or evening. It is quite rare for me to be heavily active at night, which suggests that I do not use Twitter much past midnight. This only happens very rarely and at odd times.

Language Distribution

This graph shows the language distribution of my tweets. Twitter classifies tweets into languages using language models. However, this visualization is incredibly misleading. If it measures anything, it measures the incompatibility of this model with multilingual instances of data. The graph shows how many times I tweeted in a certain language.

I do not know any languages other than English, Arabic, and French, so this is a very interesting graph! The reason for this misclassification by Twitter is that I sometimes use Romanized Arabic which is always misinterpreted as another language.

This graph shows one why NLP is a very hard task on this dataset. The content of the tweets is multilingual. One tweet can mix up to three languages and two different writing scripts. Being able to process such data without ruining its uniqueness is a very hard task. Here's an example of a tweet you can find:

"i'm genuinely so sick of uni begad تعبت, men albi te3ebt. leghayet hena w begad merci."

This tweet is a mixture of English, Arabic, Romanized Arabic (Egyptian dialect) and French. Being able to separate it without ruining the data is very hard. However, we are very close to getting meaningful analysis out of the tweet content. This is currently in the works.
